## レイヤー別の責務

### Customer Touchpoints（顧客接点）
- LP/Webサイト: 各店舗ページ、メニュー・スタイリスト紹介、SEO対策
- Web予約フォーム: 日時・スタイリスト・メニュー選択、ゲスト/会員予約
- LINE連携: 予約通知・リマインド、口コミ依頼、クーポン配信
- Google/Instagram: 口コミ管理、投稿連携、MEO対策

### Application Layer（管理画面）
- 会社管理画面: 全店舗横断ダッシュボード、売上・KPI分析、スタッフ全体管理
- 店舗管理画面: 予約カレンダー、顧客カルテ、日次業務オペレーション
- スタッフ画面: 自分のシフト・予約確認、施術記録入力、売上確認

### Backend
- Supabase PostgreSQL + RLS でマルチテナント分離
- Supabase Auth で認証
- Edge Functions で外部API連携・ビジネスロジック

### External Integrations
- LINE Messaging API: 通知・リマインド・クーポン
- Google Business API: 口コミ取得・返信
- Instagram Graph API: 投稿・コメント管理
- PayPay API: 決済連携（導入は検討中）
